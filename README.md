# Teslo API
1. Clonar el proyecto
```
git clone 
```
1. Clonar el archivo __.env.template__ y renombrar la copia a __env.__
2. Llenar las variables de entorno definidas en el archivo ```.env```
3. Levantar la base de datos - PostgreSQL
```
  docker-compose up -d
```
